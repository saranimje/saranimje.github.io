Project Title: pixel-knight-resume

Objective:
Create a gamified, story-driven resume website with pixel art, themed navigation, interactive pages (timeline, carousels, project dragons), and a light/dark mode toggle using React + Tailwind CSS.

✅ Setup Instructions
Create New Replit Project

Select React template.

Install Tailwind CSS
In terminal:

bash
Copy
Edit
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
Then in tailwind.config.js:

js
Copy
Edit
darkMode: 'class',
content: ["./src/**/*.{js,jsx,ts,tsx}"],
In index.css:

css
Copy
Edit

@tailwind components;
@tailwind utilities;
